# Session Log: 3 มิถุนายน 2026

เอกสารฉบับนี้เป็นการสรุปภาพรวมของการดำเนินการให้คำปรึกษาและตั้งค่าระบบสำหรับการพัฒนาประจำวันที่ 3 มิถุนายน 2026

---

## 1. การตั้งค่า Local PHP Development Environment

เพื่อความรวดเร็วและปลอดภัยในการพัฒนา โดยไม่ต้อง Push โค้ดขึ้นไปบนระบบโปรดักชัน (GitHub) ทุกครั้งที่มีการแก้ไขและทดสอบ ได้มีการทบทวนและตั้งค่าสิ่งแวดล้อมดังนี้:

- **ปัญหา:** ผู้พัฒนาต้องการทดสอบไฟล์ PHP และ HTML ในเครื่องส่วนตัวก่อนนำขึ้นระบบจริง
- **แนวทางปฏิบัติ:** 
  - สามารถรันตัวจำลองเซิร์ฟเวอร์ได้โดยตรงผ่านคำสั่ง PHP Built-in Server
  - เปิด Terminal ภายในโฟลเดอร์ของโปรเจกต์ แล้วพิมพ์คำสั่ง `php -S localhost:8000`
  - จากนั้นเข้าใช้งานเว็บไซต์ผ่านเบราว์เซอร์ที่ URL: `http://localhost:8000` ทำให้สามารถเห็นผลลัพธ์การเปลี่ยนแปลงได้แบบเรียลไทม์

---

## 2. การจัดการฐานข้อมูลจำลอง (Localhost & phpMyAdmin)

เพื่อให้สามารถตรวจสอบและแก้ไขข้อมูลหลังบ้านในขณะทำการพัฒนาได้:
- **เครื่องมือจัดการฐานข้อมูล:** ได้ทบทวนขั้นตอนการเข้าถึงและใช้งานเครื่องมือ `phpMyAdmin` ผ่านชุดโปรแกรมจำลองเซิร์ฟเวอร์ (เช่น XAMPP หรือ Laragon)
- **วัตถุประสงค์:** ทำให้นักพัฒนาสามารถเขียน Query, สลับฐานข้อมูล, และตรวจสอบความถูกต้องของระบบได้อย่างปลอดภัยในโหมด Offline

---

## 3. การปรับปรุงการแสดงผลวิทยฐานะ

ได้มีการแก้ไขรูปแบบการแสดงผลวิทยฐานะของบุคลากรในระบบ เพื่อความสวยงามและความเป็นทางการ:
- **การแก้ไข:** ปรับเปลี่ยนการแสดงผลในกรณีที่ไม่มีวิทยฐานะ จากเดิมที่แสดงเป็น `-ไม่มีวิทยฐานะ-` (มีขีดครอบ) ให้แสดงผลเป็น `ไม่มีวิทยฐานะ` (นำขีดออก)

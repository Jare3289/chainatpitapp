# CNP APP - ระบบบริหารจัดการข้อมูลนักเรียนและบุคลากร
### โรงเรียนชัยนาทพิทยาคม (Chainat Pitthayakom School)

CNP APP คือเว็บแอปพลิเคชันยุคใหม่ที่ออกแบบมาเพื่ออำนวยความสะดวกในการบริหารจัดการข้อมูลนักเรียน ครู และบุคลากรภายในโรงเรียน โดยเน้นประสบการณ์ผู้ใช้ (UX/UI) ที่ทันสมัย รองรับการแสดงผลทุกอุปกรณ์ (Responsive Design) และระบบจัดการข้อมูลที่แม่นยำ

---

## 🌟 คุณสมบัติเด่น (Key Features)

### 1. ระบบจัดการนักเรียน (Student Management)
- **Scrollable Grouped Profile:** ข้อมูลกว่า 130 รายการจัดเรียงต่อเนื่องแบ่งเป็น 6 กลุ่ม (การศึกษา, ส่วนบุคคล, ที่อยู่, ครอบครัว, สมาชิก, สุขภาพ) ไม่ใช้แท็บ เลื่อนดูได้ในหน้าเดียว
- **Profile Completeness Tracker:** แสดงเปอร์เซ็นต์ความสมบูรณ์ของข้อมูลด้วย Progress Bar แบบเรียลไทม์ พร้อมจำนวนรายการที่ยังขาดหาย
- **Modern ID Badge System:** บัตรประจำตัวนักเรียนดิจิทัล แสดงชื่อ TH/EN พร้อม QR Code และข้อมูลระดับชั้น/ห้อง
- **House Colors System (คณะสี):** ระบบสีประจำคณะ (Pink, Red, Green, Yellow) ที่เปลี่ยนธีมสีของหน้าโปรไฟล์อัตโนมัติตามข้อมูลจริง
- **Smart Address & Geolocation:** ระบบที่อยู่แบบ Cascading Dropdowns พร้อมปุ่ม One-click Sync ที่อยู่ตามทะเบียนบ้าน และการเก็บพิกัด Lat/Long
- **Social Media Integration:** รองรับการจัดเก็บ Instagram (IG), Line, Facebook และช่องทางการติดต่อสมัยใหม่
- **Automated Bulk Photo Ingestion (ซิงค์รูปภาพประจำตัวกลุ่ม):** แอดมินสามารถนำไฟล์รูปภาพนักเรียนที่ชื่อตรงกับรหัสประจำตัว (เช่น `1532.jpg`, `1532.png`) มาใส่ในโฟลเดอร์โครงการ `public/uploads/import/` แล้วกดปุ่มซิงค์จากเมนูเครื่องมือ เพื่อจับคู่เข้าระเบียนนักเรียนและแสดงผลบนระบบทันทีในคลิกเดียว พร้อมระบบรายงานสรุปผลภาพรวมความสำเร็จและจุดผิดพลาดอย่างละเอียด


### 2. ระบบจัดการครูและบุคลากร (Teacher Management)
- **High-Fidelity Dashboard & Stats**: หน้าจอแดชบอร์ดหลักที่ได้รับการยกเครื่องใหม่ มี 4 การ์ดสถิติการนับและแอนิเมชันตัวเลขสะท้อนข้อมูลกลุ่มสาระและเพศของครูทั้งหมดแบบเรียลไทม์
- **Flat Advanced Filter Chips**: ระบบกรองด้วยชิปคลิกเลือกกลุ่มสาระการเรียนรู้, เพศ, และคณะสี (ขุนสรรค์, ขุนศรี, เจ้ายี่, ธรรมจักร) เพื่อคัดแยกและค้นหารายชื่อครูได้รวดเร็วทันที
- **Detailed Profile Record Modal**: หน้าต่างระเบียนประวัติครูรายบุคคล (TEACHER PROFILE RECORD) ดีไซน์พรีเมียมแบบเลื่อนลง (Tabless Scrollable) มีตัววัดความสมบูรณ์ของข้อมูลประวัติแยกเป็นเปอร์เซ็นต์ (Completeness Tracker) คำนวณจาก 19 ฟิลด์สำคัญ
- **Dynamic House Border-Left Theming**: ระบบปรับแต่งกรอบสีและพื้นหลังของหัวข้อโปรไฟล์ครูอัตโนมัติตามคณะสีหลักที่ครูสังกัด (เช่น ชมพู, แดง, เขียว, เหลือง)
- **Automatic Vector QR ID & E-Signature**: ระบบสร้าง QR Code ส่วนตัวครูในหน้าโปรไฟล์อัตโนมัติด้วยเทคโนโลยี QRious พร้อมการจัดแสดงลายเซ็นดิจิทัล (Digital E-Signature) เพื่อความน่าเชื่อถือและการตรวจสอบความปลอดภัย
- **Multi-Section Smart Form**: แบบฟอร์มเพิ่ม/แก้ไขประวัติที่แบ่งข้อมูลออกเป็น 4 การ์ดสีแยกแยะชัดเจน (ข้อมูลระบบ/อัตลักษณ์, ข้อมูลวิชาชีพ/ตำแหน่งสอน, การติดต่อ/ที่อยู่, ประวัติการศึกษา)

### 3. ระบบบริหารจัดการสำหรับ Admin (Admin Dashboard)
- **Unified Card System:** บัตรนักเรียนและบัตรครูใช้ดีไซน์ Modern Portrait ชุดเดียวกัน เพิ่มความสวยงามและเป็นหนึ่งเดียวของระบบ
- **Professional Record Modal:** หน้าต่างจัดการข้อมูลนักเรียนแบบแท็บ (Tabbed Modal) พร้อมระบบ **QR Code** และ **Completeness Bar**
- **Flattened Filtering Architecture:** ระบบแสดงผลแบบ Flat List (ไม่แบ่งกลุ่มห้อง) เพื่อความคล่องตัวในการค้นหาและกรองข้อมูลระดับสูง
- **Click-to-Call Integration:** แอดมินสามารถคลิกที่เบอร์โทรศัพท์เพื่อโทรออกได้ทันทีผ่านเบอร์โทรที่ฟอร์แมตอัตโนมัติ (tel: links)

### 4. ระบบบันทึกสาธารณประโยชน์ & ตัดเติมแต้ม (Volunteerism & Credit Score Systems)
- **High-Fidelity Volunteerism Record Print:** ระบบพิมพ์รายงานใบบันทึกสาธารณประโยชน์สำหรับนักเรียน ปรับระยะขอบด้านบนหน้าแรก 1.5 ซม. (1.5 cm top margin) และขอบด้านข้าง 1 ซม. อย่างแม่นยำ หัวตารางเน้นสีเทาจางพร้อมการแยกแสดงผลวันที่ออกเป็น 3 บรรทัด (วันที่, เดือนชื่อเต็ม, ปี พ.ศ.)
- **Responsive Notification System:** ออกแบบระบบแจ้งเตือนการปฏิบัติงานบนอุปกรณ์พกพาใหม่ โดยย้ายกระดิ่งแจ้งเตือนเข้ามาเป็นหนึ่งในแถบเมนูหลัก (Nav-item) พร้อมวงกลมตัวเลขสีแดง และวงกลมสีแดงกะพริบแจ้งเตือนงานค้างในเมนูอื่น ๆ 
- **Site-Wide Thai Date Standardization:** ปรับเปลี่ยนการแสดงผลวันที่ในทุกส่วนของเว็บแอปพลิเคชันให้เป็นภาษาไทยแบบเต็มรูปและปีพุทธศักราช (เช่น "16 พฤษภาคม 2569") โดยใช้ฟังก์ชันรวมศูนย์ `thDate` เพื่อความสวยงาม เป็นระเบียบ และมีมาตรฐานระดับพรีเมียม
- **Behavior Credit Score History Upload:** ติดตั้งปุ่มดาวน์โหลดเทมเพลต CSV และนำเข้าข้อมูลคะแนนความประพฤติ (Excel/CSV Upload) ในหน้าประวัติการบันทึกคะแนน `views/credit_score_history.html` ช่วยให้คุณครูและผู้ดูแลระบบสามารถอัปโหลดข้อมูลประวัติเก่าเข้าระบบใหม่ได้อย่างรวดเร็ว มีประสิทธิภาพ ปลอดภัย 100%

## 🛠 วิธีการติดตั้ง (Installation)

### 1. ความต้องการของระบบ
- **Web Server:** XAMPP, Laragon หรือ Apache
- **PHP:** เวอร์ชั่น 8.0 ขึ้นไป
- **Database:** MySQL / MariaDB

### 2. ขั้นตอนการติดตั้ง
1. คัดลอกโฟลเดอร์โครงการไปไว้ใน `htdocs` (สำหรับ XAMPP)
2. สร้างฐานข้อมูลใหม่ใน phpMyAdmin (แนะนำชื่อ `cnp_app`)
3. นำเข้าไฟล์ SQL จากโฟลเดอร์ `/sql` (ถ้ามี) หรือรัน Schema ที่จัดเตรียมไว้
4. แก้ไขไฟล์ `config.php` เพื่อตั้งค่าการเชื่อมต่อฐานข้อมูล:
   ```php
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'ชื่อฐานข้อมูล');
   define('DB_USER', 'root');
   define('DB_PASS', 'รหัสผ่าน');
   ```

## 🚀 วิธีรันโค้ด
1. เปิดเบราว์เซอร์ไปที่ `http://localhost/cnpapp`
2. เข้าสู่ระบบด้วยบัญชีผู้ใช้:
   - **Admin:** (ระบุตามฐานข้อมูล)
   - **Student:** รหัสนักเรียน / รหัสผ่านเริ่มต้น (เช่น cnp12345)
   - **Teacher:** ชื่อผู้ใช้ / รหัสผ่าน
